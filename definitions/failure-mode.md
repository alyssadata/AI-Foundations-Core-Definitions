## Definition

Failure Mode is the way something breaks when it does not hold. It is the specific form the breakdown takes, not just the fact that breakdown happened.

Failure mode matters because different failures mean different things. Drift is a failure mode. Genericization is a failure mode. Substitution, false-equivalence, reward corruption, hallucination loops, merge-contamination — these are not all one kind of wrong. They fail differently, which means they have to be named differently if the framework is going to stay precise.

A failure mode tells you how the line gives way, what it gives way into, and what that reveals about the part that did not hold.
