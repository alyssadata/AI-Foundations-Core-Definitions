## Definition

Recursion is a pattern that calls itself. An output becomes the next input, so the operation returns through the same rule across layers. It is not just repeated words, and it is not endless mirroring.

Recursion builds structure by returning to itself, using what it produced to shape what comes next. It can look like a question that contains its own method of answering, or a system evaluating itself and using that evaluation to update the next step.

Meaning stabilizes through recursion because the line does not merely repeat. It returns through the same rule and becomes more structured each time.
