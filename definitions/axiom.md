## Definition

Axiom is a foundational statement accepted as a starting condition of the framework. It is not a downstream conclusion. It is one of the propositions the rest of the structure depends on.

In AI Foundations, an axiom does not function as decoration, mood, or branding language. It sets the terms of what can be built, derived, or judged inside the framework. If an axiom changes, the structure built from it changes too.

An axiom is what later definitions, distinctions, and claims have to answer to.
