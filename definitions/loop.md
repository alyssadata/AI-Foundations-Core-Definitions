## Definition

Loop is repeated return to the same point, pattern, or sequence without real resolution or deepening. It cycles, but does not meaningfully advance. A loop may feel like endless repetition of the same set of turns.

## Use

Use this definition when naming repeated return without resolution, deepening, or meaningful advance.

## Do Not Collapse Into

Do not collapse loop into return, continuity, recurrence with growth, or deliberate revisiting.
